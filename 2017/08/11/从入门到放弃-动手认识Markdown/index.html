从入门到放弃-动手认识Markdown

2017-08-11 Markdown语法 阅读量

Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用，不管你是程序员、文艺小青年、论坛狂热分子，还是经常收发邮件的白领。看到这里请不要被「标记」、「语言」所迷惑，Markdown的语法十分简单。常用的标记符号也不超过十几个，这种相对于更为复杂的HTML标记语言来说，Markdown可谓是十分轻量的，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。

从入门到放弃，马上动手认识Markdown语法吧！
